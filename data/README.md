# Dataset

This folder contains the competition dataset.

## Download Instructions
1. Visit the [Kaggle competition page](https://www.kaggle.com/competitions/mlp-term-2-2025-kaggle-assignment-1/data)
2. Download `train.csv`, `test.csv`, and `sample_submission.csv`
3. Place them in this directory

**Note:** Dataset files are not included in this repository due to Kaggle's terms of service.
